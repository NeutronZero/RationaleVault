# RationaleVault Memory Integration Freeze — v2.6

**Effective**: 2026-06-26
**Scope**: Memory Integration layer (H4)

---

## Architecture

The Memory Integration layer bridges the Agent Runtime with the graph-backed memory substrate.

```
Agent Applications
    ↕ MCP Tools
MCP Server v2 (v2.5)
    ↕ MCPRuntime
Agent Runtime (v2.2)
    ↕ AgentSession + Capabilities
Memory Integration (v2.6)
    ↕ MemoryBroker + MemoryContext
Memory Layer (existing)
    ↕ MemoryProvider + Retrieval + Synthesis
Knowledge Layer
    ↕ KnowledgeProvider + Synthesis
Event Ledger
```

### Key Design

| Component | Responsibility |
|-----------|---------------|
| **MemoryQuery** | Immutable query specification with type, filters, limits |
| **MemoryResult** | Scored memory with provenance chain |
| **MemoryContext** | Snapshot of query results for agent session |
| **MemoryWriteRequest** | Write request with type, importance, tags |
| **MemoryWriteResult** | Write outcome with deduplication detection |
| **MemoryBroker** | Orchestrates queries/writes through existing memory infrastructure |

## Contracts

### MemoryQuery (MQRY-[hash])

```
MemoryQuery {
    query_id: str                # MQRY-[hash]
    query_type: MemoryQueryType  # SEARCH | RETRIEVE | CONTEXT | CONTINUATION | LINEAGE
    text: str
    project_id: str | None
    memory_types: frozenset[MemoryRecordType] | None
    lifecycle_states: frozenset[MemoryLifecycleState] | None
    limit: int
    min_score: float
    include_provenance: bool
}
```

### MemoryResult (MRES-[hash])

```
MemoryResult {
    result_id: str               # MRES-[hash]
    memory_id: str
    memory_type: MemoryRecordType
    title: str
    content: str
    score: float
    lifecycle_state: MemoryLifecycleState
    source_event_ids: list[str]
    source_memory_ids: list[str]
    confidence: float
    reference_count: int
    reasons: list[str]
    retrieval_path: list[str]
}
```

### MemoryContext (MCTX-[hash])

```
MemoryContext {
    context_id: str              # MCTX-[hash]
    query_id: str
    results: list[MemoryResult]
    total_candidates: int
    retrieval_time_ms: float
    profile_used: str | None
    knowledge_context_hash: str | None
}
```

### MemoryWriteRequest (MWRT-[hash])

```
MemoryWriteRequest {
    request_id: str              # MWRT-[hash]
    memory_type: MemoryRecordType
    title: str
    content: str
    project_id: str | None
    importance: str
    tags: frozenset[str]
    source_event_ids: list[str]
    actor: str
}
```

### MemoryWriteResult (MWRS-[hash])

```
MemoryWriteResult {
    result_id: str               # MWRS-[hash]
    request_id: str
    success: bool
    memory_id: str | None
    error: str | None
    deduplicated: bool
}
```

### MemoryBroker

```python
broker = MemoryBroker()
context = broker.execute_query(query)    # Read path
result = broker.record_memory(request)  # Write path
cached = broker.get_cached_context(query_id)  # Session continuity
```
