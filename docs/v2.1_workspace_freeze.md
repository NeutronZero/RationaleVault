# v2.1 Workspace Architecture Freeze

**Date:** 2026-06-26
**Status:** FROZEN (L1 — No further modifications without ADR)
**Covers:** Epic G — Workspace (G0-G6)

## Summary

Freezes the Workspace subsystem as the public-facing interface to the RationaleVault cognitive platform. The Workspace is the boundary between human/agent interaction and the internal cognitive substrate.

## Design Invariants

1. **No I/O in projection/service:** All methods are pure functions.
2. **Deterministic IDs:** SHA-256 hashed, same inputs → same ID.
3. **Frozen dataclasses:** All domain objects are immutable.
4. **Event sourcing:** Domain Object → Event Payload → Event Ledger → Projection → State.
5. **Role-based access:** PRIMARY (read/write), ADVISOR (read/suggest), OBSERVER (read).
6. **Workspace = public face:** The workspace is the only interface humans/agents interact with.
