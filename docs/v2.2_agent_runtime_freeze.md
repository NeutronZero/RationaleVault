# v2.2 Agent Runtime Freeze

**Date:** 2026-06-26
**Status:** FROZEN (L1 — No further modifications without ADR)
**Covers:** Epic H — Agent Runtime (H0.0 + H0.1)

## Summary

Freezes the Agent Runtime subsystem as the platform layer between Workspace and external agents. The Agent Runtime owns session management, capability negotiation, workspace binding, and context package streaming.

## Design Invariants

1. **AgentProfile ≠ AgentSession:** Identity is separate from running state.
2. **Composable capabilities:** Set operations (granted ∩ ¬denied = effective).
3. **Protocol versioning:** Major version compatibility check on session creation.
4. **Session lifecycle:** CREATED → ATTACHED → STREAMING ↔ PAUSED ↔ RESUMED → DETACHED → CLOSED.
5. **Pure functions:** All methods are deterministic, no I/O.
6. **Event-sourced:** Every state change produces an event payload for ledger emission.
