# v2.3 Transport SDK Freeze

**Date:** 2026-06-26
**Status:** FROZEN (L1 — No further modifications without ADR)
**Covers:** Epic H — Transport SDK (H1.0)

## Summary

Freezes the Transport SDK as the platform layer defining HOW data moves between the Agent Runtime and external agents. Transports are stable platform concerns; vendors are extension concerns.

## Design Invariants

1. **Transports separate from vendors:** N×M → N+M. Transports define HOW, vendors define WHAT.
2. **Capabilities are composable:** `TransportCapabilities.satisfies(required)` for deterministic negotiation.
3. **Negotiation is deterministic:** Same manifest + runtime version → same outcome.
4. **Serialization is transport-level:** `WorkspacePackageSerializer` converts package → vendor-native format.
5. **Event-sourced:** Every transport state change produces an event payload.
6. **Frozen dataclasses:** All domain objects are immutable.
